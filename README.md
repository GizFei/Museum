# Museum

### 测试
   - Xiaomi MI 5s Plus(Android 7.0.0 API 24) 
   - HUAWEI EVA-AL00(Android 8.0.0 API 26)
   - Nubia NX549J (Android 6.0.1, API 23) 
   - Google Nexus 5X - 8.0 - API 26 - 1080*1920
   
### 主要功能
   - 博物馆详情⻚面五个Fragment
      - 概况（博物馆的基本信息）
      - 展览（博物馆最新新闻和展览）
      - 馆藏（传感器+音频播放+重排序）
      - 导览（3D视图容器，参考 blog.csdn.net/Mr_immortalZ ）
      - 全景（Google相关SDK）
      
   - 博物馆概况页面五个功能
      - 记录（拍照写文字进行实时记录）
      - 收藏（收藏到收藏夹）
      - 打卡（盖章动画）
      - 导航（高德相关SDK）
      - 分享（分享到QQ等社交平台）
   
### 其他特色
   - 界面美观，布局合理
   - 卡片和列表两种方式显示博物馆
